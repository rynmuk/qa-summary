# QA Tools Overview

## 1. Purpose
This document provides an overview of tools used for testing **RBAC and CRUD Users** in the Admin Dashboard.

## 2. Tools Used

### Cypress
- UI automation
- RBAC regression
- CRUD workflows

### Postman
- API testing
- Authorization validation
- Negative scenarios

### GitLab
- Test artifact storage
- Version control
- CI/CD integration

### Jira
- Bug tracking
- Sprint planning
- Test execution tracking

## 3. Tool Usage Mapping
| Activity             | Tool                  |
|----------------------|-----------------------|
| Test Case Management | Excel / GitLab        |
| Bug Tracking         | Jira                  |
| UI Automation        | Cypress               |
| API Testing          | Postman               |
| Reporting            | Confluence / Markdown |

## 4. Best Practices
- Tag automation tests with RBAC role
- Keep Postman collections modular
- Maintain traceability between test case and bug
